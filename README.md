# Amazon-Product-Market-Analysis
🚀 Project Overview
This project provides a comprehensive market analysis of Amazon products, focusing on pricing strategies, category performance, and customer satisfaction trends. The dashboard is designed to help stakeholders identify market gaps, evaluate category volume, and understand the correlation between product pricing and user ratings.

🛠 Technical Skills Demonstrated
Data Cleaning & Transformation: Used Power Query to handle messy, nested category data using Text Before Delimiter functions.

Data Modeling: Built a structured relationship model to enable dynamic filtering and slicer functionality.

Interactive Visualization: Developed a professional, grid-based dashboard using:

KPI Cards: For instant metric monitoring (Avg. Discount, Price, Rating, Product Count).

Scatter Plots: To analyze the correlation between discounted_price and rating.

Clustered Bar Charts: For category-level product volume analysis.

Business Intelligence: Designed for clarity, utilizing whitespace, consistent color schemes, and user-centric slicers.

📊 Dashboard Features
Interactive Slicers: Filter the entire dashboard by specific product categories or rating ranges.

Correlation Analysis: A scatter plot visualizes the relationship between price and satisfaction, helping to identify high-value "hidden gems."

Top N Analysis: A dynamic table showcasing the top 10 products based on user engagement.

📂 How to View This Project
To explore the interactive features, data model, and DAX measures:

Download the amazon_dash.pbix file from this repository.

Open the file using Power BI Desktop.

Interact: Use the category and rating slicers to see how the charts update in real-time.
